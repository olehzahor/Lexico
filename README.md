# Lexico

Lexico is an iOS app for building vocabulary through card-based study sessions and spaced repetition.

## What It Does

- Runs a daily learning session with unseen and due cards.
- Tracks review progress and card states (unseen, review, ignored).
- Shows session metrics (today's progress and current level completion).

## Tech Stack

- SwiftUI
- SwiftData
- Observation (`@Observable`)
